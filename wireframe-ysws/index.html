Skybase ✨


How does this works?

Skybase is a three-week program where you are challenged to ship four unique projects ranging from Game and Web Development to CAD and Hardware. These genres are presented as islands that unlock every five days, giving you just enough time to complete and ship each project successfully! By finishing each project, you earn charms that can be redeemed for stickers, swag, and more.

The islands that unlock every five days, reveals a new genre that rotates around:

  • Game Development
  • CAD
  • Web Development, and
  • Hardware!
